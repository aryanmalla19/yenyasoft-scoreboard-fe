# Yenyasoft Scoreboard Frontend

This is the frontend application for the Yenyasoft Scoreboard project, built with Vue.js.

## Prerequisites

Before you begin, ensure you have the following installed:

* [Node.js](https://nodejs.org/) (version 20 or above recommended)
* npm (comes with Node.js)
* Git

## Installation

1. **Clone the repository**

```bash
git clone https://github.com/aryanmalla19/yenyasoft-scoreboard-fe.git
cd yenyasoft-scoreboard-fe
```

2. **Install dependencies**

```bash
npm install
```

3. **Setup environment file**

```bash
cp .env.example .env
```

## Running the Application

```bash
npm run serve
```

This will start the development server. You can access the application at `http://localhost:8080` by default.
